This guide walks through getting connected to a Lunar Tear server. You will need a compatible game client and a few minutes to configure it.

## Requirements

- A device capable of running the original *NieR Re[in]carnation* client
- A copy of the client you legally obtained before the official shutdown
- Network access to the community server

:::caution
Lunar Tear does **not** distribute the game client. You must provide your own copy.
:::

## 1. Prepare your client

Make sure your client is unmodified and at a version supported by the current server build. Check the [latest release notes](/changelog/) for compatibility info.

## 2. Configure the server endpoint

Detailed steps for redirecting the client to a Lunar Tear server are maintained alongside each release. Follow the instructions in the most recent release notes — the procedure can change between versions.

## 3. Create an account

Account creation happens in-client on first connection. There is no separate signup form.

## 4. Play

You're in. Head into the Cage.

## Troubleshooting

If something doesn't work:

1. Confirm your client version matches the current server build
2. Check the [FAQ](/guide/faq/)
3. Ask in `#support` on the [Discord](https://discord.gg/MZAf5aVkJG)
