## Is this legal?

Lunar Tear is a non-commercial preservation project. The server is original work, built from clean-room reverse engineering. No game client, assets, or data are distributed — players supply their own copy.

See the [Legal Disclaimer](/guide/legal/) for the full position.

## Is it free?

Yes. No payments, subscriptions, ads, or in-app purchases. Released under the [MIT License](https://github.com/Walter-Sparrow/lunar-tear/blob/main/LICENSE).

## Will my old account or progress carry over?

No. The official servers are gone and account data is not recoverable. If you saved your own snapshot before shutdown, you can import it on your own server.

## Can I host my own server?

Yes — that's the point. Start with the [Quick Setup](/guide/setup-quick/), or jump to the [Advanced Setup](/guide/setup-advanced/) for a real deployment.

## How can I help?

- Test new builds and report what breaks
- Contribute code on [GitHub](https://github.com/Walter-Sparrow/lunar-tear)
- Help others in `#support` on [Discord](https://discord.gg/MZAf5aVkJG)
- Improve these docs where they're unclear

## Where do I report bugs?

[GitHub issues](https://github.com/Walter-Sparrow/lunar-tear/issues), or `#support` on [Discord](https://discord.gg/MZAf5aVkJG) for quick triage.
