## Is this legal?

Lunar Tear is a non-commercial preservation effort. We don't distribute the game client or any assets owned by Square Enix or Applibot — players must supply their own legally-obtained copy. The server reimplements protocols; it doesn't redistribute proprietary code.

## Is it free?

Yes. There are no payments, subscriptions, ads, or in-app purchases. The project runs on volunteer time.

## Will my old account / progress carry over?

No. The official servers are gone; account data is not recoverable.

## Is the gacha still random?

Rates and mechanics aim to mirror the original where possible. Specifics depend on the current server build — check the [release notes](/changelog/).

## Can I host my own server?

Yes — the project is open source. See the [GitHub repo](https://github.com/Walter-Sparrow/lunar-tear).

## How can I help?

- **Test** new builds and report bugs
- **Contribute code** via GitHub
- **Help others** in the Discord `#support` channel
- **Document** anything that's unclear in these docs

## Where do I report bugs?

GitHub issues on the [main repo](https://github.com/Walter-Sparrow/lunar-tear/issues), or `#bugs` on Discord for quick triage.

## Why "Lunar Tear"?

It's the flower from the NieR series — fitting for a preservation project.
