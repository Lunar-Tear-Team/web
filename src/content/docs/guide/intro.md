**Lunar Tear** is a fan-made, non-commercial preservation project for *NieR Re[in]carnation* — the mobile RPG developed by Applibot and Square Enix that shut down in April 2024.

## What this is

A private server that lets you boot the game client, log in, and play through preserved content. Built and maintained by volunteers in the community.

## What this is *not*

- An official product
- Endorsed by, or affiliated with, Square Enix or Applibot
- A distribution point for the game client or assets
- A commercial service — no payments, no IAPs, no ads

## What to expect

- Story content and character progression in playable shape
- Rough edges — this is a preservation effort, not a polished live service
- Active development; expect breaking changes between releases
