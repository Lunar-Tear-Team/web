**Lunar Tear** is a fan-made, non-commercial **preservation project** that keeps *NieR Re[in]carnation* playable after its April 2024 shutdown.

It's an open-source private server you can run yourself, on your own machine or a real host. You bring your own copy of the game client; the project provides the server side.

## What it is *not*

- An official product
- Affiliated with, endorsed by, or approved by Square Enix or Applibot
- A distribution point for the game client or its data
- A commercial service — no payments, no ads, no in-app purchases
