# Legal Disclaimer

## Project Purpose

Lunar Tear is a fan-made, non-commercial server implementation dedicated to preserving *NieR Re[in]carnation*, a mobile RPG developed by Square Enix and Applibot whose official servers shut down on April 29, 2024. The project exists solely for educational, archival, and interoperability purposes. No revenue is generated, no monetization is in place, and no commercial activity of any kind is intended or pursued.

No copyrighted game assets, binaries, compiled client code, master data, or proprietary files are distributed through this project or its repositories. Users are required to source the original game client themselves.
