# Welcome to Lunar Tear

**Lunar Tear** is a fan-made server implementation for *NieR Re[in]carnation*, the mobile RPG developed by Square Enix and Applibot, which shut down its official servers on April 29, 2024.

## How it works

Lunar Tear is **self-hosted**. There is no central server. You run the server software on your own machine and connect the original game client to it locally. Think of it like running a local emulator: everything stays on your hardware.
